# 🏋️‍♀️ Fitness-Progress-and-workout-analyzer-dashboard
📊 Overview
The Fitness Progress & Workout Analyzer Dashboard is a data visualization project designed to track and analyze fitness activities. It provides insights into calories burned, workout duration, hydration levels, BMI categories, and workout efficiency.
This dashboard helps users understand their fitness patterns and make data-driven decisions to improve their health.

#🚀 Features
🔥 Average Calories Burned
⏱ Workout Session Duration (hours)
💧 Daily Water Intake Tracking
⚖️ BMI Category Distribution
🏃 Calories Burned per Minute (by Workout Type)
🎯 Interactive filters:
Gender
BMI Category
#📌 Dashboard Insights
Calories Burned: ~905 kcal on average
Workout Duration: ~1.26 hours
Water Intake: ~2.63 liters
Workout Efficiency: Similar calorie burn across Strength, Cardio, Yoga, and HIIT
BMI Distribution: Visual breakdown of Normal, Overweight, Obese, and Underweight users
